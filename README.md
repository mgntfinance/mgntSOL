**Core benefits:**

## 🧲 mgntSOL Within the Magnet Finance Protocol

Magnet Finance is a multi-revenue deflationary DeFi protocol on Solana. Three revenue streams distribute weekly to all token holders via a two-tier system.

**The three revenue streams:**

| Stream | Source |
|---|---|
| Swap fees | Magnetic Swap via Jupiter v6 (30bps) |
| LP yield | Meteora DLMM mgntSOL/SOL pool |
| Treasury yield | Diversified yield positions |

**mgntSOL's role:** The mgntSOL/SOL Meteora DLMM pool generates LP yield that routes as a polarity amplifier to pMGNT holders every weekly Magnetic Epoch. Every SOL staked deepens protocol liquidity and strengthens the distribution system for all token holders.

**The tri-token system:**
- **$MGNT** — Governance and utility. Weekly buybacks, burns, and protocol revenue share.
- **pMGNT** — Positive polarity. Value by volume. LP yield amplifier. (Minting in Season 2)
- **nMGNT** — Negative polarity. Value by scarcity. Treasury yield amplifier. (Minting in Season 2)

Every token holder earns from the Base Layer every epoch. Polarity determines your upside.
